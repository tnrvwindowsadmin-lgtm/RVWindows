# 🧹 RVM Windows - System Cleaner

**English** | **Tiếng Việt**

---

## 📖 Overview / Tổng Quan

**RVM Windows** is a powerful system cleaning tool that helps you remove junk files, clear cache, and free up disk space on Windows.

**RVM Windows** là công cụ dọn dẹp hệ thống mạnh mẽ, giúp bạn xóa file rác, cache và giải phóng dung lượng ổ cứng trên Windows.

---

## ✨ Features / Tính Năng

| English | Tiếng Việt |
|---------|------------|
| 🗑️ Clear browser & app cache | 🗑️ Xóa cache trình duyệt & ứng dụng |
| 🧹 Remove junk files (.log, .bak, .tmp) | 🧹 Xóa file rác (.log, .bak, .tmp) |
| 📁 Delete empty folders | 📁 Xóa thư mục trống |
| ⚡ One-click clean all | ⚡ Dọn tất cả chỉ 1 click |
| 🌐 Bilingual support (EN/VN) | 🌐 Hỗ trợ song ngữ (EN/VN) |
| 📊 Real-time statistics | 📊 Thống kê thời gian thực |
| 📋 Detailed activity log | 📋 Nhật ký chi tiết |

---

## 📥 Download / Tải Xuống

**File:** `RVWindows.exe`  
**Size:** ~50 MB  
**Platform:** Windows 10/11 (64-bit)  
**Requirement:** No Python needed

**[⬇️ Download RVWindows.exe](https://github.com/tnrvwindowsadmin-lgtm/RVMWindows/releases)**

---

## 🚀 How to Use / Cách Sử Dụng

1. Download `RVWindows.exe` / Tải file `RVWindows.exe`
2. Double-click to run / Double-click để chạy
3. Click **"Scan Cache"** / Click **"Quét Cache"**
4. Click **"Delete"** to remove / Click **"Xóa"** để xóa
5. Use **"Clean All"** for full cleanup / Dùng **"Dọn Hết"** để dọn toàn bộ

---

## 🖥️ System Requirements / Yêu Cầu Hệ Thống

- Windows 10 or newer / Windows 10 trở lên
- 64-bit system / Hệ thống 64-bit
- No Python required / Không cần Python
- No admin rights needed / Không cần quyền admin
